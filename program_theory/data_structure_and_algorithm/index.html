数据结构与算法

1、概述

1.1、简介

程序 = 数据结构 + 算法

1.2、数据结构

数据结构包括线性结构和非线性结构。

  • 线性结构：数组、队列、链表和栈。
    • 特点是数据元素之间存在一对一的线性结构。
    • 线性结构有两种不同的存储结构，即顺序存储结构和链式存储结构。
  • 非线性结构：二维数组、多维数组、广义表、树、图。

2、稀疏数组

2.1、简介

当一个数组大部分元素为0或者同一个值时，可以采用稀疏数组来保存。

2.2、实现思路

  • 二维数组 -> 稀疏数组
    1. 遍历二维数组，获得有效数据个数sum。
    2. 创建稀疏数组，sum+1行3列。
    3. 稀疏数组第一行为二维数组的行，列，有效数据的个数；1到sum+1行为有效数据的行、列、值。
  • 稀疏数组 -> 二维数组
    1. 先读取稀疏数组的第一行，创建原始数组。
    2. 读取稀疏数组后面几行数据，将其赋给二维数组。

4、链表

4.1、简介

  • 有序列表，链表以节点方式存储，链式存储。
  • 每个节点包含data域和next域（指向下一个节点）。
  • 链表的各个节点不一定是连续存储。
  • 链表分为有头节点的链表和没有头节点的链表。
