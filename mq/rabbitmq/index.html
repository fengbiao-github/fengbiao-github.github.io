    1. 基本概念

    1.1. RabbitMQ

    • AMQP 协议，高级消息队列协议，用于解决跨服务调用，链接协议，不是从 API 层进行限定，而是直接定义网络交换的数据格式。
    • Erlang 语言开发，并发性好。

    1.2. 四大核心概念

    • 生产者：
    • 消费者：
    • 交换机：生产者产生的消息并不会直接发送到队列，而是将消息发送到交换机。类型有直连 direct、分发 fanout、路由 topic、标题 headers。
      RabbitMQ将消息发送到交换机，ActiveMQ将消息直接发送到队列。
    • 队列：

    1.3. 相关术语

    • Broker：消息服务器，接受和分发消息的应用，例如 RabbitMQ Server。
    • Virtual Host：虚拟主机，类似网络中的 namespace 概念。
    • Connection：生产者或消费者与消息服务器之间的 TCP 连接。
    • Channel：通道，Channel 之间完全隔离。轻量级 Connection，极大减少了操作系统建立 TCP 连接的开销。
    • Exchange：交换机。
    • Queue：队列。
    • Binding：交换机与队列之间的虚拟连接。

    2. 安装与配置

    2.1. Linux 环境

    1. 下载erlang、socat、rabbitmq-server安装包并安装。
    rpm -ivh erlang-23.1-1.el7.x86_64.rpm
rpm -ivh socat-1.7.3.2-2.el7.x86_64.rpm
rpm -ivh rabbitmq-server-3.7.18-1.el7.noarch.rpm
    1. 修改配置文件。
    # 拷贝配置文件，需注意3.8版本后rabbitmq.config.example可能不存在。
cp /usr/share/doc/rabbitmq-server-3.7.18/rabbitmq.config.example /etc/rabbitmq/rabbitmq.config
# 开放来宾用户guest访问，需取消注释配置文件中{loopback_users, []}（需要去掉最后的逗号），生产环境不建议。
# 命令行创建用户。
rabbitmqctl add_user 用户名 密码 # 添加用户
rabbitmqctl set_user_tags 用户名 标签名 # 设置用户标签
    1. 修改/etc/hosts 配置文件，并重启网关（需要在 hosts 中配置主机名）。
    systemctl restart network
    1. 关闭防火墙或者添加规则。
    systemctl start/restart/enable/disable/status firewalld
    1. 启动插件管理。
    rabbitmq-plugins enable rabbitmq_management
    1. 启动 rabbitmq-server，并配置开机自启动。
    systemctl start/restart/enable/disable/status rabbitmq-server
    1. 测试。
      RabbitMQ管理页面地址：http://ip:15672
      默认5672为客户端通信端口，15672网页通信端口，25672集群通信端口。

    2.3. 常用命令

    # 查看插件，E显式，e隐式，当隐式的插件不再被其他插件需要时，便会自动变成disable
rabbitmq-plugins list
# 开启插件
rabbitmq-plugins enable xxx
# 添加用户
rabbitmqctl add_user 用户名 密码
# 设置用户标签。
    # none：无任何角色，新建用户的角色默认为none
    # management：可以访问web界面
    # policymaker：包含management的所有权限，并可管理策略和参数
    # monitoring：包含management的所有权限，并可以看到所有连接、信道及节点相关信息
    # administrator：包含monitoring的所有权限，并可以管理用户、vhost、权限、策略、参数等
rabbitmqctl set_user_tags 用户名 标签名
# 查看系统用户
rabbitmqctl list_users
# 给用户配置配置、写、读权限
rabbitmqctl set_premissions [-p 虚拟主机] 用户名 ".*" ".*" ".*"
